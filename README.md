# CSI Club Website

This repository contains the current frontend implementation of the CSI PES MCOE website.

## Documentation

Use the docs below for full technical handover and implementation planning:

- `docs/PROJECT_HANDOVER.md` - complete system documentation, architecture, runbook, and implementation status.
- `docs/STACK_MIGRATION_BLUEPRINT.md` - stack-agnostic plan for rebuilding or extending into full stack.

## Quick Start

```sh
npm install
npm run dev
```

Default local URL: `http://localhost:8080`

## Scripts

- `npm run dev` - start development server
- `npm run build` - create production build
- `npm run preview` - preview production build
- `npm run lint` - run ESLint
- `npm run test` - run Vitest once
- `npm run test:watch` - run Vitest in watch mode
